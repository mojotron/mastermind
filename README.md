# Mastermind with Ruby

Mastermind or Master Mind is a code-breaking game for two players. 
Learn more on [Wikipedia](https://en.wikipedia.org/wiki/Mastermind_(board_game)).

In this project there are two options to play. First, player is Code Breaker and tries 
to guess a secret code generated by the program. Second option, player is Code Maker, makes 
a secret code for the program to guess with player simulator. The program makes AiAiCodeBreaker 
class, that is not real AI, but simulates player by randomly guessing colors and memorizing where 
are red and white flags. Game has 12 turns and 6 colors for players to choose. Game logic is build 
around checking and comparing secret code and player input.

Game is won if all 4 red flags are found. Red flag can only be marked if both position and color 
are correct. White flag can only be marked if color matches but it is  on wrong position and that
position is not already marked with red flag

Board and tokens are made with UTF-8 characters, for better UI.
